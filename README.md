# Classic-Snake-Game

## Description
A classic Snake Game developed using Python and the Turtle graphics library. The game allows users to control the snake, collect food, increase score, and avoid collisions with walls and the snake body.


## Technologies Used
 1.Python
 2.Turtle Module
 3.Random Module
 4.Time Module

## Features
- Snake movement using keyboard controls
- Random food generation
- Score tracking system
- Snake body growth functionality
- Wall collision detection
- Self collision detection
- Automatic game reset after collision

## Learning Outcomes
- Improved Python programming skills
- Understanding of game logic implementation
- Event handling using keyboard controls
- Collision detection concepts
- Working with Turtle graphics
